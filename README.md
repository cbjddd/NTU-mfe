# NTU MFE 参考资料

南洋理工大学金融工程硕士课程参考资料库

## 简介

本仓库收集整理了南洋理工大学金融工程硕士(Master of Financial Engineering)项目的学习资料，包括课程笔记、作业、项目和参考文献等。

## 仓库结构

```
NTU-mfe/
├── Year1/
│   ├── Semester1/
│   │   ├── Course1/
│   │   ├── Course2/
│   │   └── ...
│   └── Semester2/
│       ├── Course1/
│       ├── Course2/
│       └── ...
├── Year2/
│   ├── Semester1/
│   └── Semester2/
├── Resources/
│   ├── Books/
│   ├── Papers/
│   └── Tools/
└── README.md
```

## 使用说明

- 资料按学年、学期、课程分类整理
- 每个课程文件夹包含相关的笔记、作业、项目等材料
- Resources 文件夹包含通用的参考书籍、论文和工具

## 贡献指南

1. Fork 本仓库
2. 创建新的分支 (`git checkout -b feature/add-materials`)
3. 添加或更新资料
4. 提交更改 (`git commit -am 'Add: [课程名称] 相关资料'`)
5. 推送到分支 (`git push origin feature/add-materials`)
6. 创建 Pull Request
