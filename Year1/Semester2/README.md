# Year 1 Semester 2 - 第一学年第二学期

## 课程列表

请为每门课程创建独立的文件夹，建议命名格式：`[课程代码]-[课程名称]`

### 典型课程可能包括：
- 数值方法 (Numerical Methods)
- 风险管理 (Risk Management)
- 投资组合理论 (Portfolio Theory)
- 金融计量经济学 (Financial Econometrics)

## 文件组织建议

每个课程文件夹应包含：
- `notes/` - 课程笔记
- `assignments/` - 作业和练习
- `projects/` - 课程项目
- `exams/` - 考试相关资料
- `references/` - 参考资料
