# Year 1 Semester 1 - 第一学年第一学期

## 课程列表

请为每门课程创建独立的文件夹，建议命名格式：`[课程代码]-[课程名称]`

### 典型课程可能包括：
- 随机微积分 (Stochastic Calculus)
- 金融市场与机构 (Financial Markets and Institutions)
- 固定收益证券 (Fixed Income Securities)
- 衍生品定价 (Derivatives Pricing)

## 文件组织建议

每个课程文件夹应包含：
- `notes/` - 课程笔记
- `assignments/` - 作业和练习
- `projects/` - 课程项目
- `exams/` - 考试相关资料
- `references/` - 参考资料
