# Year 2 Semester 1 - 第二学年第一学期

## 课程列表

请为每门课程创建独立的文件夹，建议命名格式：`[课程代码]-[课程名称]`

### 典型课程可能包括：
- 高级衍生品 (Advanced Derivatives)
- 信用风险建模 (Credit Risk Modeling)
- 算法交易 (Algorithmic Trading)
- 金融工程项目 (Financial Engineering Project)

## 文件组织建议

每个课程文件夹应包含：
- `notes/` - 课程笔记
- `assignments/` - 作业和练习
- `projects/` - 课程项目
- `exams/` - 考试相关资料
- `references/` - 参考资料

对于大型项目，建议创建详细的项目文档和代码组织结构。
