# Year 2 Semester 2 - 第二学年第二学期

## 课程列表

请为每门课程创建独立的文件夹，建议命名格式：`[课程代码]-[课程名称]`

### 典型内容可能包括：
- 毕业论文 (Dissertation)
- 高级选修课程
- 实习报告 (Internship Reports)
- 职业发展资源 (Career Development)

## 文件组织建议

### 毕业论文
建议创建 `Dissertation/` 文件夹包含：
- `proposal/` - 论文提案
- `literature_review/` - 文献综述
- `data/` - 数据文件
- `code/` - 分析代码
- `drafts/` - 论文草稿
- `final/` - 最终版本

### 其他课程
每个课程文件夹应包含标准的组织结构：notes, assignments, projects, exams, references
